## 🖥️ What jido_symphony does

jido_symphony helps turn a project idea into working software. It uses a set of agents to break a job into smaller tasks, work on them, and merge the results into one codebase.

It is made for long-running project work, where one task leads to the next. The app can:

- Set up a new project from a short description
- Create a task list for the work
- Send tasks to separate agents
- Keep each agent in its own workspace
- Merge finished work into the main branch
- Continue until the task list is done

## 🛠️ Typical workflow

The app follows a loop:

1. Read the current task list.
2. Pick the next open item.
3. Give the item to an agent.
4. Let the agent make changes in its own workspace.
5. Merge the changes into the main branch.
6. Check for merge conflicts.
7. Resolve conflicts if needed.
8. Move to the next item.

This keeps the work moving without asking you to manage each step by hand.
